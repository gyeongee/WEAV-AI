# Cloudflare Pages 프론트엔드 체크리스트 (Vite)

## 사전 점검
- [ ] 프론트 경로가 `frontend/`에 존재한다
- [ ] SPA 리다이렉트 파일 존재: `frontend/public/_redirects`
- [ ] 기본 헤더 파일 존재: `frontend/public/_headers`
- [ ] `frontend/.env`에 `VITE_API_BASE_URL`이 설정되어 있다 (터널/백엔드 도메인)

## 로컬 빌드 (선택 검증)
```bash
cd frontend
npm ci
npm run check:env
npm run build
```
- [ ] 빌드 결과가 `frontend/dist/`에 생성되었다

## Pages 프로젝트 설정 (Git 연동)
- [ ] Root directory: `frontend`
- [ ] Build command: `npm run build`
- [ ] Build output directory: `dist`

## 수동 업로드 (Git 미사용 시)
- [ ] `frontend/dist/` **폴더 내부의 파일들**을 업로드한다 (폴더 자체가 아님)

## 빠른 스모크 테스트
- [ ] `https://<your-project>.pages.dev` 접속 정상
- [ ] SPA 라우팅 정상 (깊은 경로 직접 접속 테스트)
- [ ] JS/CSS의 `Content-Type`이 올바르게 반환된다 (`application/javascript`, `text/css`)
